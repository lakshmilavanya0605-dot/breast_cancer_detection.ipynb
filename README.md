This project predicts whether a tumor is malignant or benign using Logistic Regression.

- Data Collection
- Data Preprocessing
- Model Training
- Accuracy Evaluation
- Prediction System

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📊 Result
Achieved good accuracy in predicting breast cancer diagnosis.
